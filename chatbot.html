Asistente AI del Mundial

Base para integrar un chatbot que responda sobre horarios, sedes, plantillas, jugadores y predicciones.

Estado del desarrollo

Esta página deja preparada la zona de asistente. La integración real necesitaría un backend o una función serverless para proteger la clave del modelo de IA.

Usuario: ¿Cuándo juega España?
Asistente AI: Consultaría el calendario actualizado y te devolvería fecha, hora en España, rival y fase.
Usuario: Dame un pronóstico razonado.
Asistente AI: Analizaría forma reciente, ranking, plantilla, bajas y estadísticas ofensivas/defensivas.

Arquitectura recomendada

Frontend

GitHub Pages con HTML, CSS y JavaScript.

Backend ligero

Cloudflare Worker, Vercel Function o Spring Boot si quieres ampliarlo.

IA

Modelo pequeño para resúmenes y un motor estadístico para probabilidades.
